# YouTube Clone

A responsive YouTube homepage clone built using HTML and CSS. This project was created to practice web development fundamentals including layouts, positioning, Flexbox, and CSS Grid.

## Features
- Responsive YouTube-style homepage layout
- Navigation sidebar
- Video thumbnails section
- Channel profile pictures
- Header with search bar and icons
- CSS Flexbox and Grid implementation

## Technologies Used
- HTML5
- CSS3

## Concepts Practiced
- Flexbox
- CSS Grid
- Positioning
- Responsive Design
- Styling Components

## Folder Structure
- channel-pictures/
- icons/
- intro-to-html/
- styles/
- thumbnails/

- youtube.html
